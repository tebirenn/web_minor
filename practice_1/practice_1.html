Плюсы:

  1. VS Code позволяет разрабатывать как консольные приложения,
    так и приложения с графическим интерфейсом
  2. В редакторе присутствуют встроенный отладчик, инструменты для работы с Git и средства рефакторинга,
    навигации по коду, автодополнения типовых конструкций и контекстной подсказки
  3. Большим плюсом редактора является поддержка большого количества языков,
    таких как C++, C#, Python, PHP, JavaScript и других

Минусы:

  1. При работе с отдельными языками как python, java и тд слабый
    функционал по сравнению с редакторами от компаний JetBrains
